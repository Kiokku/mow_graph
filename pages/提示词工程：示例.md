- ## 项目结构和工作流程
	- ```markdown
	  graph TD
	      A[命令行输入 npm run wuxia] --> B[index.js入口]
	      B --> C[cli解析参数]
	      C --> D{选择Chain}
	      D --> |chain1| E1[基础提示词]
	      D --> |chain2| E2[改进提示词]
	      D --> |chain3| E3[拆分提示词]
	      
	      subgraph chain3流程
	          E3 --> F1[代码分析]
	          F1 --> F2[武侠设定]
	          F2 --> F3[情节设计]
	          F3 --> F4[生成注释]
	      end
	      
	      B --> F[processCodeFile]
	      F --> G[读取代码文件]
	      G --> H[生成注释]
	      E1 --> H
	      E2 --> H
	      F4 --> H
	      H --> I[写回文件]
	      I --> J[输出耗时]
	  ```
